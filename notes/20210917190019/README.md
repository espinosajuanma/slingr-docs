# Format dates with `sys.utils.time`

Use `sys.utils.time.format()` to format a date.

```js
var formatedDate = sys.utils.time.format(new Date(), 'MM-dd-yyyy');
log(formatedDate);
```

- Date - `object`
- pattern - `string`
- timezone - `object`
